# 케마 KEMA - 방문요양기관 교육 관리 서비스

방문요양기관의 요양보호사 교육을 효율적으로 관리하는 웹 애플리케이션입니다.

## 🎯 주요 기능

### 1. 센터장 관리자 페이지 (`director.html`)
- **요양보호사 관리**: 등록, 수정, 삭제 기능
- **요약 대시보드**: 전체 보호사 수, 이수 완료 인원 표시
- **상세 정보 관리**: 이름, 연락처, 생년월일, 자격증 번호, 입사일, 상태 관리

### 2. 본사 통합 관리 페이지 (`headquarters.html`)
- **기관 통합 관리**: 모든 가입 기관 목록 조회
- **보호사 현황**: 기관별 소속 보호사 수 확인
- **알림톡 발송 설정**: 기관별 맞춤 발송 요일 및 시간 설정

## 📁 프로젝트 구조


## 🎨 디자인 시스템

### 색상 팔레트
- **Primary**: 청록색 계열 (#0EA5E9, #0284C7)
- **Success**: 녹색 계열 (#10B981)
- **Danger**: 빨간색 계열 (#EF4444)
- **Warning**: 주황색 계열 (#F59E0B)

### 주요 스타일 특징
- **Glassmorphism**: 반투명 배경 + 블러 효과
- **그라데이션**: 부드러운 색상 전환
- **애니메이션**: 호버 효과, 페이드 인/아웃
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 대응

## 🚀 시작하기

### 실행 방법

1. 프로젝트 폴더로 이동:
```bash
cd c:\개발폴더\Work\Kema
```

2. `index.html` 파일을 브라우저에서 열기:
   - 파일 탐색기에서 `index.html`을 더블 클릭
   - 또는 브라우저 주소창에 파일 경로 입력

3. 원하는 페이지로 이동:
   - **센터장 관리자**: `director.html`
   - **본사 통합 관리**: `headquarters.html`

### 개발 서버 실행 (선택사항)

Python이 설치되어 있다면 간단한 로컬 서버를 실행할 수 있습니다:

```bash
# Python 3
python -m http.server 8000

# 브라우저에서 http://localhost:8000 접속
```

## 📊 데이터베이스 스키마

프로젝트는 다음 7개의 테이블로 구성됩니다:

### 핵심 테이블
1. **organizations** - 기관 정보
2. **caregivers** - 요양보호사 정보
3. **education_contents** - 교육 콘텐츠
4. **completion_records** - 이수 결과

### 알림톡 관련 테이블
5. **notification_templates** - 알림톡 템플릿
6. **notification_settings** - 기관별 발송 설정
7. **notification_logs** - 발송 이력

자세한 스키마는 `database_schema.md` 파일을 참조하세요.
